# Analisis de datos sobre la educacion en la Alcaldia Tlahuac Ciudad de Mexico

Se realiza un análisis de datos sobre la educación en la alcaldía Tlahuac de la Ciudad de México en los , obteniendo información como los talleres con mayor actividad, edad promedio de la población así como su nivel académico, entre. Estos datos se obtuvieron del programa social y educativo PILARES que ha tomado relevancia por ofrecer cursos, talleres gratuitos, las personas continuen con sus estudios entre otras actividades mas.  

## Tecnologias implementadas

-Python
-Excel
-Tableau

## Arquitectura

CSV -> Extract -> Transform -> Load -> CSV clean -> Tableau
