# An-lisis-de-datos-sobre-la-educaci-n-en-la-Alcald-a-Tlahuac-Ciudad-de-M-xico
En este proyecto realice un análisis de datos sobre la educación en la alcaldía Tlahuac de la Ciudad de México, obteniendo información como los talleres con mayor actividad, edad promedio de la población así como su nivel académico, entre otras mas. Estos datos se obtuvieron del programa social PILARES.
